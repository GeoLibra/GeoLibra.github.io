requests

如果下载比较大的文件，requests可以实现将下载的文件分块写到磁盘。

wget

wget是一个从网络上自动下载文件的自由工具。它支持HTTP，HTTPS和FTP协议，可以使用HTTP代理。

wget常用参数：

  • -c 断点续传
  • -t BUMBER 最大尝试连接次数(–tries=NUMBER)
  • -T SECONDS 响应超时秒数 (–timeout=SECOBDS)
  • -P PREFIX 将文件保存在目录(–directory-prefix=PREFIX)
