User-Agent

2019-01-04 HTTP 阅读次数 0

文章目录

  1. 浏览器标志
  2. 操作系统标志

User Agent，用户代理，是HTTP协议中的一部分。是一种向访问网站提供你所使用的浏览器类型及版本、操作系统及版本、浏览器内核等信息的标识。通过这个标识，用户所访问的网站可以显示不同的排版从而为用户提供更好的体验或者进行信息统计。

浏览器标志

由于历史上的浏览器大战，当时想获得图文并茂的网页，就必须宣称自己是 Mozilla 浏览器。此事导致如今User-Agent里通常都带有Mozilla字样。
